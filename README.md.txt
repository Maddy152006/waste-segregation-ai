# AI-Based Waste Segregation ♻️

## Overview
This project uses MobileNetV2 to classify waste into:
- Metal
- Paper
- Plastic
- Trash

## Accuracy
- ~88% validation accuracy

## Tech Stack
- TensorFlow / Keras
- Transfer Learning

## Project Structure
- model/ → trained model
- notebooks/ → training code
- src/ → prediction code (to be added)

## Future Work
- Camera integration
- Robotic sorting system
